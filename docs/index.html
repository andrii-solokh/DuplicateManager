Duplicate scanning

Queueable Apex jobs scan records with Salesforce native Datacloud.FindDuplicates support.

Record review

Filter duplicate sets by object, inspect matching records, and track scan status from one Lightning dashboard.

Field-level merge

Compare duplicate records side by side and choose the values to keep before merging.

Account record page workflow

Use the merge experience for focused Account cleanup when users only need to compare and merge account duplicates.

Merge account duplicates from a record page

Duplicate Manager can support a focused Account record page workflow for teams that do not need the full dashboard. Start from an Account record, review detected duplicates, choose the master account, and keep the best field values.

Install the latest promoted package

Version 1.5.1.2 is promoted for production Salesforce orgs. Assign the Duplicate_Manager permission set after installation. 

sf package install --package 04tgK0000009cxJQAQ --target-org your-org-alias --wait 10
sf org assign permset --name Duplicate_Manager --target-org your-org-alias
